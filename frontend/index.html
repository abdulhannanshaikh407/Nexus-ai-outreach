Features

🔍

Research Agent

Automatically researches prospects using company data, LinkedIn profiles, and industry trends.

✉️

Email Writer

Generates highly personalized cold emails using proven frameworks like AIDA and PAS.

🛡️

Spam Optimizer

Analyzes and optimizes email content to avoid spam filters and improve deliverability.

🧠

FAISS Memory

Stores past interactions in a vector database for smarter personalization.

📊

CSV Bulk Send

Process hundreds of leads at once with CSV import and automated bulk sending.

🚀

Dry Run Mode

Test your entire pipeline without sending real emails to validate content and flow.

Tech Stack & Setup

Python 3.11+ LangChain OpenAI FAISS FastAPI Docker

Quickstart

1

Clone the repository

git clone https://github.com/abdulhannanshaikh407/cold-email-system.git
cd cold-email-system
2

Configure environment

cp .env.example .env
# Add your OPENAI_API_KEY to .env
3

Run with Docker

docker-compose up
